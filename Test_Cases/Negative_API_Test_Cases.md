# Negative & Edge Case API Test Cases

1. Verify API returns 404 for incorrect endpoint.
2. Verify API returns 400 for malformed request body.
3. Verify API returns error when Content-Type header is missing.
4. Verify API handles empty request body gracefully.
5. Verify API returns appropriate error for unsupported HTTP method.
6. Verify API does not expose sensitive data in error response.
