# Postman Collections – API Testing Project

This folder contains Postman execution details for the API Testing Project.

## Tool Used
- Postman Desktop Application

## Environment
- Environment Name: ReqRes_Env
- Base URL: https://reqres.in

## APIs Executed
- GET – List Users
- POST – Create User
- PUT – Update User
- PATCH – Partial Update User
- DELETE – Delete User

## Validations Covered
- Status code verification
- Response body validation
- Response time validation
- Positive and negative scenarios

## Note
Postman collections and environments can be shared during interviews or project walkthroughs.
