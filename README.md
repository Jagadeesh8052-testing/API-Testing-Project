## API Testing Project Details

This repository includes a complete API testing project using Postman.

### Coverage
- API Test Plan
- Test Cases for all HTTP methods:
  - GET
  - POST
  - PUT
  - PATCH
  - DELETE
- Negative API test cases
- Postman collections documentation

### Tools & Technologies
- Postman Desktop App
- REST APIs
- JSON
- GitHub

### Purpose
This project demonstrates real-world API testing skills including request validation, response verification, and proper documentation.
